Reached through local adapters — no cloud relayer
RailCall is bring-your-own-key and local-first. Each integration is reached through a small set of generic local adapters, so your data goes straight from your machine to the provider — through the Airlock, and receipted. Unlike Prismatic, Composio, or Merge, no third-party cloud sits in the middle. You set each key on your own machine (env var, 0600); RailCall's cloud never sees or stores it. A connector turns green only when a real key + a live run mint a signed receipt — reachable code isn't "done" until it's proven.
